HTML TAGS/DECLARATIONS

Tag Name Description
<!DOCTYPE html> DOCTYPE Ensures that the browser renders the page in standards mode, which provides a consistent rendering across different browsers
<img> Image HTML tag used to embed images in web pages. It is a self-closing tag, meaning it doesn't require an end tag
<p> Paragraph Any body of text that isn't a heading (h1-h6). You cannot have a <p>element as a child of the <figcaption> element
